# MatchGame
Projeto Integrador 2 de Computação

## Descrição
- Aplicação web de encontros e registros de partidas esportivas, com sistemas de recompensas e ranqueamentos para incentivar a contínua prática de esportes.

## Funcionalidades
### Criação de Partidas
- O sistema de criação de partidas permite você primeiramente escolher o tipo de partida convidar outro usuário para representar o outro time, e assim começar a convidar outros usuários para preencher os times, ao final da partida os dois representantes precisam confirmar o resultado do jogo.

### Perfil
- No perfil de usuário será mostrado além das informações básicas que compõem o perfil do usuário informações de partidas participadas e seus resultados, esportes preferidos, posições preferidas nesses esportes e etc...

### Recompensas
- O sistema contará também com um sistema de moedas virtuais, o que permite a compra de componentes para a personalização do perfíl como títulos, cores para o perfil e afins.
- Também contará com um sistema de experiência e níveis, para destacar a habilidade deste indivíduo.
- Para o usuário conseguir ganhar experiẽncia ele precisa vencer partidas, caso perca a partida ele perde experiência, a recompensa é calculada de acordo com o placar, quantidade e nível dos adversários, quanto mais equilibrado melhor a recompensa.
- Para conseguir essas moedas o usuário pode ganhar os jogos ou interagir na comunidade, nos jogos a recompensa é calculada de acordo com o placar, quantidade e nível dos adversários, quanto mais equilibrado melhor a recompensa.
- É possível também apostar as moedas virtuais nos jogos, a aposta é com base na vitória ou derrota, a recompensa é distribuida para quem apostou e não para o time inteiro.
