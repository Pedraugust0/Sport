# Requisitos do sistema

## Funcionais
### Conta
1\) Criar Conta
  - O usuário deve poder criar conta utilizando: nome de usuário, email, senha.
  - O sistema deve ser capaz de checar com REGEX o email e checar o domínio para ver se o email é valido e se já não existe uma conta com esse email em uso.
  - A senha deve possuir pelo menos 8 chars.

2\) Logar no site
  - O usuário deve poder entrar no site usando seu email e senha para autenticação.
  - Caso as informações estejam incorretas ele deve receber um aviso para caso não tenha uma conta poder criar uma.
  - Caso ele esqueça a senha dele, ele deve poder informar qual o email da conta dele para receber um código de confirmação de identidade para resetar sua senha.
