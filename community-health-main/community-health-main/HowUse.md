# Como usar
guia basico para rodar o projeto no seu computador.

## pre-requisitos

- node.js (versao 16 ou superior)
- npm ou yarn

## passo a passo

### 1. clonar o repositorio

```bash
git clone <url-do-repositorio>
cd community-health
```

### 2. instalar dependencias

```bash
npm install
```

ou se preferir usar yarn:

```bash
yarn install
```

### 3. rodar o projeto

```bash
npm run dev
```

ou com yarn:

```bash
yarn dev
```

### 4. acessar no navegador

apos executar o comando acima, o projeto estara disponivel em:

```
http://localhost:5173
```

se a porta 5173 estiver em uso, o vite automaticamente tentara outra porta (5174, 5175, etc).

## tecnologias utilizadas

- react
- vite
- tailwindcss
- lucide-react (icones)
